Hexo搭建博客

发表于 2015-09-08   |   分类于 Hexo   |  
1. 安装Github。
2. 安装node.js。
3. 安装Hexo（Github选择 Git bash）,输入命令：
npm install -g hexo
4. 创建hexo文件夹(如C:\Hexo)，在（Git bash）中键入如下命令初始化hexo文件夹。
# 进入 C:\Hexo 
cd C:\Hexo
# 初始化hexo文件夹
hexo init
# 安装依赖包
npm install
5. 本地验证

现在我们已经搭建起了hexo本地博客，在（Git Bash）,进入目录（C:\Hexo），执行如下语句

# 生成本地静态页面
hexo generate
# 开启本地hexo服务
hexo server

此时在浏览器中输入 localhost:4000 就可以看到已经搭建起来的本地hexo博客
