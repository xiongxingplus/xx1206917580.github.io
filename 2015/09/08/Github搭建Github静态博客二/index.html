Github搭建Github静态博客二

注册Github账号

已有账号的直接登录即可，没有账号的请在此注册。

创建repository

在自己的GitHub主页的右下角，点击New repository 按钮创建一个新的repository，项目名为username.github.io。

设置SSH keys

  1. 在（Git Bash）下键入如下命令（邮箱是你注册Github时候的邮箱），

    ssh -keygen -t rsa -C “username@example.com”

  2. 然后键入以下命令

    ssh-agent -s

  3. 继续输入命令

    ssh-add ~/.ssh/id_rsa

  4. 之后，添加SSH key到Github里面。

  5. 最后在（Git Bash）下执行如下命令，验证和github是否联通：

    ssh -T git@github.com

弹出如下提示，则说明已经和github建立了连接

$ ssh -T git@github.com
Hi patton199! You've successfully authenticated, but GitHub does not provide she
ll access.
部署

编辑_config.yml(C:\hexo下)。配置deploy选项

deploy:
  type: git
  repository: git@github.com:xx1206917580/xx1206917580.github.io.git
  branch: master

然后，执行下列命令完成github部署

# 生成html静态页面
hexo generate
# 将静态页面发不到github上去
hexo deploy

现在在浏览器中输入username.github.io就能访问我们的博客了。
